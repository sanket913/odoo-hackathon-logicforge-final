# FlowForge ERP + ERPilot AI

Furniture manufacturing ERP for Shiv Furniture Works.

## Structure

- `frontend/` - React frontend application
- `backend/` - Express API, Prisma schema, and ERP services

## Local Run

Backend:

```bash
cd backend
npm install
npm run prisma:generate
npm start
```

Frontend:

```bash
cd frontend
npm install
npm run dev
```
